# Bible_Search_App
A simple and lightweight Bible search application built using **HTML, CSS, and JavaScript**.   The app allows users to search for Bible verses by passage and choose from multiple translations using the **Bible API**. It features a clean, responsive, and easy-to-use interface.


2. Open the application  
   - Open the `index.html` file in your web browser  
   - No installation or API key is required

3. Use the app  
   - Enter a Bible passage (e.g., `John 3:16` or `Genesis 1:1`)  
   - Select a Bible translation  
   - Click **Search** to view the verse
